# TaskTracker Pro

A Java OOP project for managing tasks.

## Concepts Used
- Encapsulation
- Inheritance
- Abstraction
- Polymorphism
- Interfaces
- File Handling

## Features
- Login system
- Add tasks
- View tasks
- Update task status
- Admin role

## Run

Compile:
```bash
javac Main.java
```

Run:
```bash
java Main
```

```text
TaskTrackerPro/
├── Main.java

```
